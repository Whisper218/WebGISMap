锦绣禾下——全国主要农产品产量分布

2024年全国各省份主要农产品产量统计(数据来源：国家统计年鉴 港澳台地区暂未统计)



地图展示地址：https://webgismap.vercel.app/

📖 项目简介 (Overview)

本项目是一个基于 Web GIS 技术的 3D 数据可视化交互平台。通过对 2024 年中国 31 个省份（不含港澳台） 的 9 大类主要农产品（粮食、棉花、油料、麻类、糖类、烟叶、蚕茧、茶叶、水果）产量数据进行可视化，利用 3D 柱状图直观展示我国农业产值的空间分布格局与区域差异。

✨ 核心功能 (Features)

- 多维空间展示：采用 3D 柱状图技术，高度映射产值规模，颜色区分作物类别。
- 九大类农产品切换：支持粮食、茶叶、水果等 9 种农产品的动态筛选与数据渲染。
- 交互式地理查询：点击省份要素可弹出详细的产值看板及横向对比图表。
